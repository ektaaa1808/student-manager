# student-manager
student management system in C++
A simple C++ project to manage student records using file handling.

## 🚀 Features
- ➕ Add Student
- 📋 Display Students
- 🔍 Search Student
- ❌ Delete Student
- 💾 Data stored in file (persistent)

## 🛠 Tech Used
- C++
- File Handling (fstream)

## ▶️ How to Run
g++ main.cpp -o main
./main
